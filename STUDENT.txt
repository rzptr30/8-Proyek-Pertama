CARA_UJI_PUSH=
1) Login agar token tersimpan.
2) Buka menu Tentang. Pastikan Base URL: https://story-api.dicoding.dev/v1.
3) Klik "Enable Push" → di Network muncul POST /v1/notifications/subscribe (status 200/201).
4) Klik "Uji Notifikasi Lokal" → notifikasi muncul dari SW.
5) Klik "Disable Push" → di Network muncul DELETE /v1/notifications/unsubscribe (OK).

CARA_UJI_OFFLINE=
1) Production build (dist) atau GitHub Pages.
2) Buka beranda + detail + halaman #/saved.
3) DevTools → Application → centang "Offline" → reload → shell aplikasi tetap tampil (bukan laman dino).
Catatan: SW melakukan warm-up index.html pada install agar offline dev/production berjalan.

CARA_UJI_INDEXEDDB=
1) Buka detail cerita → klik "Simpan".
2) Application → IndexedDB → stories-db → savedStories berisi entry.
3) Buka #/saved → tampil item yang disimpan (bukan semua story).
4) Klik "Hapus" → entry hilang dari UI dan IndexedDB.
5) Offline → #/saved tetap menampilkan data.
